TaskFlow lets teams create projects, break them into tasks, and track who's doing what.
When you sign up, you either get Admin or Member access. Admins set everything up — they create the projects, decide who's on each one, and can see the full picture across the whole team. Members see only the projects they've been added to and the tasks assigned to them.
Creating a task takes about 10 seconds. You give it a title, pick which project it belongs to, assign it to someone, set a priority (low, medium, high), add a due date if needed, and that's it. You can also write a longer description if the task needs more context.
Every task has a status — To Do, In Progress, or Done. You can change it directly from the task table without opening anything, just a quick dropdown. The task list has filter tabs at the top so you can see only what's in progress, only what's done, and so on.
Overdue tasks show up with a red dot so nothing slips through quietly.
The dashboard gives you a quick read on how things are going — how many tasks are done, how many are still open, how many are overdue, and a progress bar for each project showing completion percentage.
There's also a Team page (admin only) where you can see everyone who's signed up, change their role between Admin and Member, or remove them from the workspace.
